## Where things stand

## Decisions made

- **`.bito.yaml` is the canonical config name** — all docs updated from `.bito-lint.yaml`. bito-lint v0.3.0 supports both, but `.bito.yaml` is shorter and matches the actual file in this repo.
- **`.justfile` deleted** — fully superseded by `bito-lint lint` with path-based rules. CHANGELOG updated to remove justfile mention.
- **`claylo-marketplace` is the primary install path** — `claude plugin add claylo-marketplace/building-in-the-open` is first in README; manual `git clone` is the fallback.
- **Codename: "Four Writers Walk Into a Repo"** — release postcard and notes in `dist/`.

## What's next
